
INSTRUCCIONES PARA USAR LA APLICACIÓN

1. Asegúrate de tener instalado Python 3 en tu computadora. Puedes descargarlo aquí: https://www.python.org/downloads/

2. Instala Flask, que es el servidor web simple que usaremos. Para eso abre la consola (cmd) y ejecuta:

   pip install flask

3. Guarda la contraseña de aplicación que generaste de Gmail y pégala en el archivo 'app.py' en la línea que dice:

   APP_PASSWORD = 'TU_CONTRASEÑA_DE_APLICACION'

   Reemplaza 'TU_CONTRASEÑA_DE_APLICACION' por la contraseña que generaste.

4. Abre la consola (cmd), navega a la carpeta donde descomprimiste los archivos y ejecuta:

   python app.py

5. Abre tu navegador y entra a la dirección:

   http://127.0.0.1:5000/

6. Escribe tu correo Gmail en la página y presiona "Mostrar último correo".

7. Verás el último correo recibido en esa cuenta.
